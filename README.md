A good utility for simple and customized loading of textures into OpenGL for subsequent use.

## Gradle
```groovy
repositories {
    maven {
        name = "ferra13671-maven"
        url = "https://ferra13671.github.io/maven/"
    }
}

dependencies {
    implementation "com.ferra13671:gl-texture-utils:latest" //Replace 'latest' with the version of GLTextureUtils you need.
}
```

## Example
For example check out `example/src/main/java/com/ferra13671/gltextureutils/example/Main.java`.
